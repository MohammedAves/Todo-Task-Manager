Todo-Task-Manager

Overview

The Todo Task Manager is a simple and efficient web application built using Spring Boot for the backend and JSP for the frontend. It allows users to manage their tasks seamlessly with features to add, edit, delete, and view tasks. The application provides a clean interface where tasks are displayed in a table format, showing details like task name, due date, and status. Action buttons for editing and deleting tasks make task management quick and intuitive.

Features

- Add new tasks with a name, date, and status.

- Edit existing tasks to update their details.

- View a list of all tasks in a structured table.

- Delete tasks that are no longer needed.

Technologies Used

- **Backend**: Spring Boot

- **Frontend**: JSP (Java Server Pages)

- **Database**: MySQL 

How to Run

Clone the repository:

```bash
git clone <repository-url>

Navigate to the project directory:

cd todo-task-manager

Configure the database in the application.properties file.

Build and run the project using Maven:

mvn spring-boot:run

Open your browser and access the application at:

http://localhost:8080
